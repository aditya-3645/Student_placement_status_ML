# Dataset overview 
The dataset contains 50,000 student records with logically connected academic, technical, and soft-skill attributes that influence campus placement outcomes. Each feature follows real-world educational patterns such as CGPA impacting skills, skills impacting internships, and all collectively influencing placement status.

## Columns 
Student_ID, Age, Gender, Degree, Branch, CGPA, Internships, Projects, Coding_Skills, Communication_Skills. Aptitude_Test_Score, Soft_Skills_Rating, Certifications, Backlogs, Placement_Status
